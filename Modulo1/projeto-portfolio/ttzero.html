Projeto: Validação de Freios TT Zero

Tecnologia Usada: C++

Projeto feito como projeto final em grupo para o University of Nottingham racing team. Feito completamente em C++ o projeto foi extensivo com as formulas dos calculos de calor e dissipaçao de calor obtidos de jornais academicos, os dados foram extraidos de corridas passadas. O README do projeto é bem detalhado, estaria honrado se você desse uma olhada!
